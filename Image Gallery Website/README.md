# Image Gallery Website 🖼️

A responsive and visually appealing Image Gallery website developed during my **Frontend Developer Internship at SyntecxHub**. This project focuses on creating a clean gallery interface with organized image layouts, responsive design, and modern frontend styling techniques.

---

## 🚀 Project Overview

The Image Gallery project was built to practice frontend development concepts by designing a modern gallery interface that displays images in a structured and responsive layout. The project emphasizes UI design, responsiveness, and user-friendly browsing experience.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5

---

## ✨ Features

- Responsive image gallery layout
- Clean and modern UI
- Organized image sections
- Mobile-friendly design
- Bootstrap grid system implementation
- Hover effects and styling
- Smooth visual presentation
